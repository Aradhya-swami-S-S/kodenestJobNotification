# KodNest Premium Build System

A calm, intentional design system for serious B2C products.

## Design Philosophy

Calm, Intentional, Coherent, Confident. Not flashy, not loud, not playful.

## Installation

```bash
npm install
npm run dev
```

## Design Principles

- Color System: 4 colors maximum (#F7F6F3, #111111, #8B0000, muted accents)
- Typography: Serif headings, sans-serif body, 16-18px, line-height 1.6-1.8
- Spacing: 8px, 16px, 24px, 40px, 64px (no random values)
- Interactions: 150-200ms transitions, ease-in-out, no bounce

## Global Layout Structure

Every page follows:
[Top Bar] → [Context Header] → [Primary Workspace + Secondary Panel] → [Proof Footer]
